Project created thanks to the Programación en español, it is a navigation app using the google maps api, using technologies such as React for the front-end and Laravel for the back-end

Among its features are:
* Record start and end journeys.
* Show the route on a map
* Calculate the distance and price of the journey

## Available Scripts

### `npm start`

Runs the app in the development mode.\

The page will reload when you make changes.\
You may also see any lint errors in the console.
